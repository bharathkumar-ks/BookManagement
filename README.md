Workflow (BookService)

Start Application – Run BookMain and the menu appears in the console.

Add Book – User enters Book ID, Title, Author, and Copies. The system stores the book and sets available copies.

Register Member – User enters Member ID and Name. The system adds the member.

Borrow Book – User enters Member ID and Book ID. If the book is available and rules are satisfied, a borrow record is created and available copies decrease.

Return Book – User enters Borrow Record ID. The system updates the return date and calculates fine if overdue.

Search Book – User searches by title or author and matching books are displayed.

Reports – View borrowed books by member, book availability, and top borrowers.

Exit – Application closes (data is stored only in memory).
